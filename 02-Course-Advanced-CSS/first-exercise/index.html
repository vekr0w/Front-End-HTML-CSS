This website is created as an exercise given from the Soft Uni advanced CSS course

The course "CSS Advanced" takes a deeper look at the modern Front-End design and developement. The participants in the course would have the chance to enrich their knowledge in creating Web Pages by applying animations, icons and grid for dynamic design that applies to devices with all types of screen sizes(Desktop, Tablets, Mobile phones). An important skill for developing and following an effective working process is the use of modern instruments for improving productivity, the participants would have the chance to learn exactly that by applying them into practical project. The final grade for the course is formed based on homework and practical exam.

Source: CSS Advanced - November 2019
